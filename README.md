## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm, yarn, pnpm or bun

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/wasu101/my-profile.git
cd my-profile
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
# or
pnpm install
```

3. **Run the development server**
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

4. **Open your browser**
```
http://localhost:3000
```
